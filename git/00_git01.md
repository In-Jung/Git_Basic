   ``` LS
   00_git01.md  LOL/  a  c  coc  git.md  images/
   ```

   `ls -al` : 상세 목록 조회

   ```total 21

2.  `cd` :폴더 변경 

   1. `cd` : ~ 홈 폴더로 이동
   2. `cd ..`  : 이전, 상위 폴더로 이동

3. `mkdir`  : 폴더 생성

4. `touch` :  파일 생성

5. `cp`  : 복사    `cp a b`   a라는 파일을 복사해 b 생성

6. `mv` :  이동  `mv a.txt b/`    `b/`는 폴더명
